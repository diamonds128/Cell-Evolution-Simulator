# Cell Evolution Simulator / 细胞演化模拟器

This is a Scratch project about cellular evolution, built with TurboWarp.
这是一个关于细胞演化的Scratch项目，使用TurboWarp制作

## ℹ️ Overview / 项目概述

This project simulates the process of cellular evolution.
本项目模拟了细胞演化的过程

The primary reason for publishing it on GitHub is to implement an automated update-checking feature.
将其发布到GitHub的主要目的是为了实现自动检查更新的功能

Along with that, other related project files are also provided here.
同时，也提供了其他相关的项目文件

## 🚀 How to Run / 运行方法

**Important:** This project was **not** created using the default Scratch editor.
**重要提示：** 本项目**并非**使用默认的Scratch编辑器创建

It was built with **TurboWarp** and utilizes some of its exclusive extensions.
它是使用 **TurboWarp** 构建的，并利用了一些其独有的扩展功能

**To ensure proper functionality, you must run it using the latest version of [TurboWarp](https://turbowarp.org).**
**为确保功能正常，您必须使用最新版本的 [TurboWarp](https://turbowarp.org) 来运行它**

1.  Download the project file (`*.sb3`) from this repository.
    从本仓库下载项目文件 (`*.sb3`)

2.  Open it in the **TurboWarp** editor or player.
    在 **TurboWarp** 编辑器或播放器中打开它

3.  Click the green flag to start.
    点击绿旗开始运行

**Alternative Methods / 其他运行方式：**
- You can also download the pre-packaged HTML file, which should run in most modern web browsers.
  你也可以直接下载已经打包完毕的HTML文件，理论上大部分现代浏览器都能运行它
- There may be plans to package it as an executable (EXE) file in the future.
  后续也有可能打包成exe
